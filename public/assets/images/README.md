# Images Folder

Place your image files here. Supported formats: JPG, PNG, WebP, SVG

## Recommended Image Specifications:

### Hero Images:
- **Format**: WebP or JPG (with fallback PNG)
- **Resolution**: 1920x1080 or higher
- **Aspect Ratio**: 16:9 (for hero sections)
- **File Size**: Optimize for web (< 500KB recommended)

### Research Page Images:
- **Format**: WebP or JPG
- **Resolution**: 1200x800 or higher
- **Aspect Ratio**: 3:2 or 16:9
- **File Size**: < 300KB recommended

### Icon/Logo Images:
- **Format**: SVG (preferred) or PNG
- **Resolution**: Vector (SVG) or 512x512 (PNG)
- **File Size**: < 100KB

## Usage in Components:

### Research Page Images:
- Robotics: `/assets/images/robotics-hero.jpg`
- AI: `/assets/images/ai-hero.jpg`
- etc.

## Image Optimization Tips:
1. Use tools like ImageOptim, TinyPNG, or Squoosh to compress images
2. Use WebP format for better compression (with JPG/PNG fallback)
3. Consider using Next.js Image component for automatic optimization
4. Create responsive image sizes for different screen sizes
